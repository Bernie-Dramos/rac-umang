# Rotaract Club of Umang Website

Welcome to the official repository for the Rotaract Club of Umang website!

## Overview

This website serves as the digital home for the Rotaract Club of Umang, showcasing our projects, members, and contact information. It is designed to be fully responsive and accessible on all devices.

## Features

- **Responsive Design:** Adapts to all screen sizes, including mobile devices.
- **Hamburger Menu:** Easy navigation on smaller screens.
- **Hero Section:** Prominent introduction with club mission.
- **Projects & Members:** Dedicated pages for club activities and leadership.
- **Contact Page:** Get in touch with the club.

## Technologies Used

- HTML5
- CSS3 (with [Tailwind CSS](https://tailwindcss.com/))
- JavaScript (vanilla)
- [Favicon](assets/rotary-logo.png) for branding
